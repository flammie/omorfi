# ` ORDINAL `

Ordinal, lexical feature based on UD NumType, refer to http://universaldependencies.org/u/feat/NumType.html

## Default formats
| Omorfi | Apertium | FTB 3.1 | Giella |
|:------:|:--------:|:-------:|:------:|
| ` [NUMTYPE=ORD]` | ` %<ord%>` | ` % Ord` | ` +Ord`  |
